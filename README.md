Quote Generator

A simple, static web application that displays random inspirational quotes when a button is clicked.
Built with plain JavaScript, HTML, and CSS, focusing on clarity, performance, and ease of use.

Live Demo

https://neth1122.github.io/Quote_generator/

Tech Stack

HTML

CSS

Vanilla JavaScript

Features

Generates random quotes instantly

Lightweight and fast

Fully client-side

Responsive layout

No external APIs or frameworks

Privacy & Security

This project does not collect or store any user data.
There are no forms, authentication flows, downloads, or redirects.
All functionality runs entirely in the browser using static files.

Project Structure
Quote_generator/
├── index.html
├── style.css
└── script.js

Local Setup

To run the project locally:

git clone https://github.com/neth1122/Quote_generator.git
cd Quote_generator
open index.html


No build steps or dependencies are required.

Purpose

This project was created to practice core JavaScript concepts such as DOM manipulation, event handling, and simple UI logic in a real-world, user-facing example.
